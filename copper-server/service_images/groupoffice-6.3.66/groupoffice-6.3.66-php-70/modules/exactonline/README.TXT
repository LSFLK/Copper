1. Ga naar App center om app aan te maken: https://apps.exactonline.com/nl/nl-NL/Manage
2. Klik op registreer een nieuwe API key
3. Vul app naam in
4. Vul Redirect URI in http://[DOMAIN]/index.php?r=exactonline/oauth/callback
5. Klik op OK en vervolgens weer op bewerken en kopieer de Client Secret en Client ID.
6. Vul deze en de gebruikersnaam in bij Projecten -> Beheer -> Exact online en klik op Toepassen
7. Klik op "GET TOKEN". 
8. Sta de app toe en klik op ok. Als het goed is krijg je de melding dat exact online connected is.
9. Vindt je division code: https://support.timewax.com/hc/en-us/articles/225509167-Exact-Online-Where-do-I-find-the-division-code-
10. Vul de division code bij de juiste project template in.
11. Vindt Artikel UID (bv 6b014e74-bfd1-4ba8-be9b-2649d6b94f65) uit link bij Verkoop -> overzicht artikelen in exact online.
12. Stel deze in bij instellingen
13 stel ook journal/dagboek in. "70" bij dg. "50" bij testlokatie. Je kan het dagboek in exact zien als je bij verkoopfactuur op via settings icoon rechtsboven op "aanpassen" klikt en dan "dagboek" aanzetten.
